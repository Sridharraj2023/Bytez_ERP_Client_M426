# React Frontend - Digital Agency ERP

## Quick Start

```bash
cd client
npm install
npm start
```

App runs on http://localhost:3000

## Features

- React 18 with Hooks
- React Router v6
- React Bootstrap UI
- Axios API integration
- JWT Authentication
- Protected Routes

## Pages

- Login
- Dashboard
- Clients (CRUD)
- Projects (CRUD)
- Tasks (CRUD)
- AI Tools

## Build

```bash
npm run build
```
